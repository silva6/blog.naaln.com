Docker 初步试玩

发表于 2016-05-19 | 分类于 技术 |

昨天调试完Codeship和EC2的连接后开始尝试，连接Docker

Docker之所以有用，是因为把代码从一个机器迁移到另一个机器经常是困难的。它尝试去使得软件迁移的过程变得更加可信和自动化。Docker 容器可以移植到所有支持运行 Docker 的操作系统上。
